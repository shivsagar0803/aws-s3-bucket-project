# AWS S3 Bucket Creation and Object Upload

## 📌 Project Overview
This project demonstrates how to create an Amazon S3 bucket and upload an object using the AWS Management Console. It is a beginner-level project to understand basic cloud storage concepts.

---

## 🚀 Steps Performed

1. Logged into AWS Management Console  
2. Opened Amazon S3 service  
3. Created a new S3 bucket with a unique name  
4. Selected region (Asia Pacific - Mumbai)  
5. Kept default security settings (Block Public Access enabled)  
6. Uploaded a file (object) into the bucket  
7. Verified object URL and checked access permissions  

---

## 📂 Project Output

- Successfully created an S3 bucket  
- Uploaded a file into the bucket  
- Verified that the object is private (Access Denied when accessed via URL)  

---

## 🧠 Key Learnings

- What is Amazon S3 (Simple Storage Service)  
- Understanding of buckets and objects  
- Basics of object storage  
- Default security (private access) in S3  
- Importance of access control in cloud
